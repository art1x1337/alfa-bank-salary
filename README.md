Альфа Солюшн — KPI Tracker

Приложение для отслеживания KPI и продаж, с авторизацией и статистикой.

Работает как PWA на Node.js + Express + SQLite, полностью фронт на HTML/CSS/JS, поддержка iOS.

------------------------------------------------------------------------------------------------

Функционал


Регистрация и логин пользователей (JWT)

Просмотр и редактирование продуктов, учёт продаж

Цель по доходу и визуализация прогресса

Суммарная статистика по дню и месяцу

Копирование отчёта в буфер обмена

PWA: адаптивный интерфейс для iOS/Android

Запуск через локальный сервер Node.js

---------------------------------------------------------------------------------------------------

kpi-tracker/
├─ public/               # фронтенд (HTML/CSS/JS)
├─ server.js             # backend, роуты, авторизация
├─ db.js                 # SQLite + инициализация таблиц
├─ package.json
└─ README.md

---------------------------------------------------------------------------------------------------

Настройка и запуск

Клонируем репозиторий:
git clone <YOUR_REPO_URL>
cd kpi-tracker
npm install

Запуск:
npm start

------------------------------------------------------------------------------------------------------

Проверка работы

Регистрация и логин нового пользователя
Добавление/удаление продаж по продуктам
Ввод цели по доходу и проверка прогресса
Копирование отчёта в буфер обмена


--------------------------------------------------------------------------------------------------------
